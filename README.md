# Zoneflow

Visual Zone Workflow Engine

## Packages

- @zoneflow/core
- @zoneflow/react

## Apps

- playground

## Development

### Install
```bash
pnpm install
```

### Build
```bash
pnpm build
```

### Run Playground
```bash
pnpm --filter playground dev
```

### Build Packages
```bash
pnpm --filter @zoneflow/core build
pnpm --filter @zoneflow/react build
```
